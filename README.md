## Installation

Afin d'installer et de lancer mon projet, clonez ce-denier grâce à la commande :

```
git clone https://github.com/CorentinLeGallic/git-evaluation_groupe-5.git
```

Puis, créer le fichier `.env` à la racine du dépôt et ajouter les variables suivantes :

```
JWT_KEY="SECRET_JWT_KEY" # Une clé JWT de votre choix
NODE_ENV="DEV"
```

Enfin, lancez le projet grâce à la commande :

```
npm run start:dev
```
